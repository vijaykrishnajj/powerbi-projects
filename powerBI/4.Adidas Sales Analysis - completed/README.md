This project presents an end-to-end Power BI dashboard developed to analyze Adidas sales performance. It delivers actionable insights into revenue, profitability, regional performance, and product trends, enabling stakeholders to make informed, data-driven decisions.

🎯 Objectives

Monitor overall sales and operating performance

Analyze seasonal trends and monthly sales patterns

Evaluate regional and state-level performance

Identify top-performing products and retailers

📊 Key Features

📈 Interactive dashboards with dynamic date and region filters

🎯 KPI cards for real-time business performance tracking

🗺️ Geospatial analysis using map visualizations

🔍 Drill-down capabilities for deeper insights

🎨 Clean, intuitive, and user-friendly dashboard design

📌 Key KPIs

💰 Total Sales: 899.90M

📊 Operating Profit: 332.13M

📦 Units Sold: 2M

📈 Operating Margin

📊 Dashboard Insights

📅 Monthly sales trend analysis to identify seasonality and peak periods

🌍 State-wise sales distribution using map visuals

🧭 Regional performance analysis (West, Northeast, Southeast, South, Midwest)

🛍️ Product category insights to identify top-performing segments

🏬 Retailer performance analysis (West Gear, Foot Locker, Amazon, Walmart, etc.)

🔍 Key Findings

Identified peak sales periods and seasonal demand patterns

Highlighted high-performing regions and states

Discovered top-selling product categories

Evaluated key retailers driving major revenue
