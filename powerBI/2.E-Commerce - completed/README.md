🍽️ Swiggy Restaurant Analysis Dashboard (Power BI)
🚀 Project Overview

This project features an interactive Power BI dashboard built using the Swiggy restaurant dataset. It provides deep insights into restaurant performance, customer preferences, and market trends across multiple cities, enabling smarter business decisions.

🎯 Key Features

🌆 City and area-level analysis with dynamic filters and drill-down capabilities

📊 Interactive visualizations for exploring restaurant data in real time

🍴 Insights into pricing, ratings, delivery performance, and restaurant distribution

🗺️ Map-based visuals to analyze geographical trends

🎨 Clean and user-friendly dashboard design for effective storytelling

📊 Key Metrics Analyzed

⭐ Average Customer Ratings

💰 Average Price / Cost for Two

⏱️ Average Delivery Time

🏪 Restaurant Density

📍 Location-wise Performance

🔍 Insights Generated

Identified high-demand locations with dense restaurant activity

Discovered popular food categories across cities

Highlighted premium pricing zones and customer preferences

Analyzed delivery efficiency to improve service performance

🛠️ Tools & Technologies

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Data Modeling Techniques

Interactive Dashboard Design

🔄 Data Processing Workflow

Imported and connected Swiggy dataset into Power BI

Cleaned and transformed raw data using Power Query

Built relationships and structured the data model

Created calculated measures and KPIs using DAX

Designed interactive visuals and filters for exploration

🎨 Dashboard Highlights

Dynamic city and area filters for granular analysis

Combination of charts, maps, and KPI cards

Drill-down capabilities for deeper insights

Focus on clarity, usability, and visual storytelling

💡 Business Impact

Helps identify top-performing locations and restaurants

Supports decisions on pricing strategies and expansion areas

Improves understanding of customer behavior and preferences

Enables data-driven operational improvements
