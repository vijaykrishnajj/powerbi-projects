Overview

Developed an interactive Power BI dashboard to analyze Indian Premier League (IPL) season performance. The dashboard allows users to explore season-wise metrics, team performance, and match outcomes through intuitive visualizations and interactive reporting.

Objectives 🎯

Track the number of matches played per season.

Identify season-wise match winners and team rankings.

Aggregate innings data for season-wise analysis.

Provide stakeholders with actionable insights into team performance and IPL trends.

Key Features 📊

Matches Played Per Season – Bar chart visualizing the number of matches per season.

Season-Wise Match Winners – Table displaying winners, seasons, and rankings.

Sum of Innings by Season – Bar chart showing total innings aggregated season-wise.

Visual Branding – IPL team logos and a winners list banner for thematic presentation.

Skills Demonstrated 🛠️

Data Modeling – Integrated and shaped IPL datasets (season, matches, winners).

DAX Calculations – Created measures like Count of match_id and Sum of innings.

Data Visualization – Built bar charts, tables, and custom-themed visuals in Power BI.

Dashboard Design – Arranged interactive elements and applied IPL branding for clarity and engagement.

Data Analysis – Interpreted season-wise trends and match outcomes to provide insights.

Business Value 💡

Enables stakeholders to analyze team performance and match statistics.

Supports reporting and strategic decisions for IPL-related analytics.

Provides a visually engaging, interactive experience for exploring season-wise trends.
