🚀 Overview

This project presents an end-to-end Power BI dashboard developed to analyze supermarket sales data. It delivers actionable insights into revenue, profit, customer behavior, and branch performance, enabling data-driven decision-making for business optimization.

🎯 Objectives

Monitor overall sales and profitability metrics

Analyze customer behavior across different demographics

Evaluate branch-level performance

Track sales and profit trends over time

📊 Key Features

📈 Interactive dashboards with dynamic slicers and filters

🎯 KPI cards for real-time performance tracking

🧩 Drill-down capabilities for detailed analysis

📊 Transaction-level reporting for granular insights

🎨 Clean, intuitive, and user-friendly dashboard design

📌 Key KPIs

💰 Total Sales: 61K

📊 Total Profit: 57.77K

📦 Total Quantity: 1K

🧾 Total Tax: 2.89K

📊 Dashboard Insights

👩‍🦰 Gender-wise analysis of sales and profit distribution

🌆 City-wise performance comparison using pie and donut charts

🏪 Branch-level performance analysis (Branch A, B, C)

📅 Monthly sales and profit trend analysis

🧾 Detailed transaction insights (invoice, sales, profit, ratings, quantity)

🔍 Key Findings

Identified top-performing branches and cities

Discovered customer purchasing patterns by gender

Highlighted monthly sales trends and fluctuations

Provided insights into transaction-level performance metrics
